🎓 Student Grade Tracker (Java)

📌 Project Description

The Student Grade Tracker is a simple Java-based console application that allows users to input and manage student grades. It calculates key statistics like average, highest, and lowest scores, and displays a summary report.

---

🚀 Features

- 📥 Input student names and marks
- 📊 Calculate:
  - Average marks
  - Highest score
  - Lowest score
- 📋 Display all student details
- 🧠 Uses ArrayList for dynamic data storage
- 💻 Console-based interface

---

🛠️ Technologies Used

- Java
- ArrayList (Java Collections)
- OOP Concepts

---

▶️ How to Run

1. Install Java (JDK)
2. Open terminal / command prompt
3. Compile the program:
   javac StudentGradeTracker.java
4. Run the program:
   java StudentGradeTracker

👉 If using online compiler (like Programiz), make sure class name is Main
